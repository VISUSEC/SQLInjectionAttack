# VISUSEC - Visual Web Security Education Platform

**VISUSEC（ビジュセック）** は、サイバー攻撃を「視覚的」に体験し、
身近な脅威の解像度を高めることを目的とした Web セキュリティ演習ツールです。

座学だけではイメージしづらい攻撃手法（XSS、SQL インジェクション等）を、
擬似的な環境で **安全に体験・実験** することができます。

本リポジトリは **VISUSEC プロジェクトを構成するリポジトリの一つ**です。

**Webサイト:** https://visusec.jp
